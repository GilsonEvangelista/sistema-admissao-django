# Sistema Web de Admissão

Sistema web desenvolvido em **Django + PostgreSQL** para gerenciamento do processo de admissão de candidatos.

## Funcionalidades

- Cadastro de candidatos
- Registro de exames médicos
- Avaliação psicológica
- Teste de aptidão física (TACF)
- Entrevista
- Painel administrativo Django

## Tecnologias utilizadas

- Python
- Django
- PostgreSQL
- HTML / CSS / JavaScript
- Chart.js (futuro dashboard)

## Estrutura do sistema

Candidato  
├── Exame Médico  
├── Psicologia  
├── TACF  
└── Entrevista
